Parses and constructs new Metadata object from given view over EXIF data.

Parameters
exif_dataView over EXIF data. Must be at least the size of this object plus all required offsets.
relativeWhether the offsets in exif_data are relative to the start of the view. If this flag is set, parser will assume that first byte in exif_data is the start of needed data. Otherwise, constant JPEG_HEADER_SIZE will be used as the start of needed data. Default: false.
optionsOptional decoder options. Default: default options.
Returns
Metadata object with parsed data.
